# gh-actions-workshop

This is a GitHub Actions Workshop for the USDC Software Engineering Huddle! We will be covering the following concepts:
- Basics of Github Actions: workflows, event triggers, jobs, runners, and steps
- Variables & Passing Data in GitHub Actions: environment variables, built-in GitHub environment variables, inputs, outputs
- Using Actions from the GitHub Marketplace: Custom actions vs Marketplace actions, tokens & permissions, how to read docs and use them in your workflows
- Use Case: Developing a CI/CD Pipeline: Setting up Node & Python in runners, deploying to GitHub pages
- Use Case: Repository & Organization Management: Using GitHub Actions for org-wide management and software inventory reporting

## Getting Started
Fork the repository to start! This will be your playground for creating, running, and testing workflows.

**Exercise 4**
For this exercise, you will be creating a CI/CD pipeline for the React app in this repository. Use the commands below to run the app.

```
npm install # Install dependencies
npm ci # Clean install, good for prod environments
npm run dev # Run this locally
npm run build # Create a build
```

## Examples
This workshop contains 3 example workflows for your reference. Run these workflows and read the logs to get a better understanding of their functionality. These will be useful for completing the exercises!

## Exercises 
This workshop has 4 exercises for you to complete, applying concepts learned from the presentation. Based on your experience level and learning style, feel free to complete them however you'd like, in whatever order you'd like. Send up your answer as a PR to this upstream repository and we will do a little show & tell on your solution OR if you need help, we can pair program together.
